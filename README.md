# Chicken Tracking Demo

A Streamlit application for real-time chicken tracking using YOLOv8n and modern MOT trackers.

## Features

- Upload your own video (MP4, AVI, MOV, MKV)
- Choose from three trackers: ByteTrack, OC-SORT, BoT-SORT
- Side-by-side display of original and processed video
- Live chicken count displayed on the processed video
